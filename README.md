# Portfolio - Yellamandala Sameer

A professional portfolio website for placement purposes, built with HTML, CSS, and JavaScript.

## Features

- **Modern Design** — Dark theme with cyan accents
- **Responsive** — Works on desktop, tablet, and mobile
- **Smooth Animations** — Scroll-triggered reveal effects
- **All Resume Content** — Education, projects, skills, achievements, leadership

## Quick Start

1. Open `index.html` in your browser, or
2. Use a local server (e.g. `npx serve .` or Live Server in VS Code)

## Deployment

- **GitHub Pages**: Push to a repo and enable GitHub Pages
- **Netlify / Vercel**: Drag & drop the folder or connect your repo
- **Any static hosting**: Upload the files as-is

## Customization

- Edit `index.html` to update content
- Edit `styles.css` to change colors (see `:root` variables)
